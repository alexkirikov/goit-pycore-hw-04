# goit-pycore-hw-04
This project contains four independent Python tasks.  
Below is a short description and instructions for running each task.

---

## **Task 1 — total_salary(path)**
Reads a salary file and calculates total and average salary.

**Data file:** `data/salary.txt`
Alex Korp,3000
Nikita Borisenko,2000
Sitarama Raju,1000

**Run:**
python task1.py

---

## **Task 2 — get_cats_info(path)**
Reads a file with cat data and returns a list of dictionaries.

**Data file:** `data/cats.txt`

**Run:**
python task2.py

---

## **Task 3 — Directory Tree Visualizer**
Uses `colorama` to print colored directory structure.

**Virtual environment:**
python -m venv venv
venv\Scripts\activate 
pip install colorama

**Run:**
python task3.py data/picture

---

## **Task 4 — Console Assistant Bot**
Supports commands:
`hello`, `add`, `change`, `phone`, `all`, `exit`.

**Run:**
python task4.py
